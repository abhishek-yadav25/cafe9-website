# Cafe Website (Django)

A full-stack cafe website developed using Django framework.

## Features
- Home, About, Menu, Services and Contact pages
- Dynamic website using Django
- Admin panel for content management
- Responsive design

## Technologies Used
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite

## Run Project

1. Install requirements
pip install -r requirements.txt

2. Run server
python manage.py runserver
